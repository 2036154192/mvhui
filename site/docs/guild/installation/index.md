# 安装

## 版本

muui 目前还处于快速开发迭代中。

## 使用包管理器

**我们建议您使用包管理器（如 NPM、[Yarn](https://classic.yarnpkg.com/lang/en/) 或 [pnpm](https://pnpm.io/)）安装muui**，然后您就可以使用打包工具，例如 [Vite](https://vitejs.dev/) 或 [webpack](https://webpack.js.org/)。

```javascript
# 选择一个你喜欢的包管理器

# NPM
$ npm install mvhui --save

# Yarn
$ yarn add mvhui

# pnpm
$ pnpm install mvhui
```
